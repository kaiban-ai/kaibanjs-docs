# Ecosystem

The **Ecosystem** section highlights community projects that build on KaibanJS—runtimes, tooling, and integrations that are **not** part of the core framework maintained under [github.com/kaiban-ai](https://github.com/kaiban-ai).

:::caution[Third-party software]

Projects listed here are **community contributions**. They are **not** officially supported by the KaibanJS core team unless explicitly stated. For bugs, features, and security issues, use each project’s own repository and maintainers.

:::

## Featured projects

| Project | Description |
|--------|-------------|
| [Kaiban Distributed](/ecosystem/kaiban-distributed) | Distributed, actor-style runtime for KaibanJS with Redis/BullMQ or Kafka, HTTP JSON-RPC gateway, live board sync, and optional MCP/A2A-style APIs. |

## Suggest a project

If you maintain or use a KaibanJS-related project that could help others, [open an issue](https://github.com/kaiban-ai/KaibanJS/issues) or reach out on [Discord](https://kaibanjs.com/discord). We may add it here when it is clearly documented, actively maintained, and a good fit for the community.
